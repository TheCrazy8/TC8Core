# TC8Core

TC8Core is the shared core library for TheCrazy8's Minecraft modding projects and modpack. It contains reusable systems, utilities, content, and APIs that are intended to be used across multiple mods rather than being tied to a single project.

## Features

- Shared code and utilities
- Common blocks and items
- Reusable registration helpers
- Foundation for future TC8 mods
- Built for NeoForge 1.21.1

## Development

- **Minecraft:** 1.21.1
- **Loader:** NeoForge
- **Java:** 21

## Goals

TC8Core exists to reduce duplicated code between mods and provide a consistent framework for future development. As the project grows, it will include common APIs, gameplay systems, registries, and other reusable components.
